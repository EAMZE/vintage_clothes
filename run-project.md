# Run front end
> After git pull, it is required to install new packages added to the project using: `npm install`
> then run this command `npm run dev` to start the frontend

# Run backend (db.json) based on json server
> in a different terminal run this command: `npm run server` to run the json-server..
> You will see a list of endpoints to use in ur frontend app. example: `http://localhost:3001/products`
